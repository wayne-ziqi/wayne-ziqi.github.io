快速开始

0. 下载

git clone https://github.com/wayne-ziqi/face-detect

1. 准备环境

如果你只需要用到人脸检测的功能，可以跳过此步（因为人脸检测是由前端导入blazeface_1000e模型直接实现的）。 而如果需要使用人脸标记的功能，你需要准备python运行环境（python >= 3.8）。如果你的环境中未安装Paddlepaddle， 那么你需要先导入Paddlepaddle，教程在这里 

pip install -r requirements.txt

2. 运行脚本

cd到face_landmark文件夹下，执行face_landmark.py。同样，如果你只需要用到人脸检测，只需要在浏览器中打开 index.html文件即可 

python face_landmark.py

3. 上传文件并检测

我们首先需要上传一张图片，点击开始检测和开始标记执行识别程序。检测结果会给出每张人脸的位置坐标和置信度 如果人脸标记无法识别到人脸，那么浏览器会弹出alert提示。

4. 效果展示

5. 2.0版本新增综合标记

在2.0版本中我们基于tensorflow的预训练模型和tensorflow.js框架增加了综合检测的功能，即将人脸检测和关键点标记在一张图片中呈现遗憾的是，当前模型似乎仅支持单人脸标记，但是标记效果相对更优，并会给出检测到的五官名称（并且不需要配置python环境）。 因此如果你只需要检测单一人脸且需要更多的关键点标记，欢迎尝试！
